**Project Title**

Mini software developer job scraper - Python

**Overview**

Scrapes software engineering job postings from KPMG, LLoyds, and NHS websites, storing them in a database

**Technologies Utilised**

Python
requests
BeautifulSoup
JSON APIs
SQLite

**Notes**

This was a test project to practice and demonstrate a few subskills.

I initially planned this to be a more useful tool that scrapes software engineering graduate schemes, but abandoned that for a few reasons (may start this project again in the future). 

I acknowledge that there are many engineering issues and areas of improvement for this project, including:

- Testing
- Logging
- Error handling
- Resilience to changes in HTML structure
- etc

During future projects, where I'm building something of value and interest, I'll incorporate these and other software engineering principles.
